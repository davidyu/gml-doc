v

v: Float32Array

Inherited from Matrix.v

  • Defined in mat.ts:11

The raw contents of the matrix, encoded as a Float32Array. Note that this is stored in row-major order, so it cannot be directly passed into WebGL uniform methods. Use the matrix.m property, which returns a transposed Float32Array.

toMat4

  • toMat4(): Mat4
    • Defined in 3d/mat3.ts:189

    constructs a Mat4 with the contents of this Mat3 forming the top-left portion of the new Mat4. The translation portion of the new Mat4 is assumed to be zero.

    E.G.:

    a b c       a b c 0
d e f  -->  d e f 0
g h i       g h i 0
            0 0 0 1

    Returns Mat4
